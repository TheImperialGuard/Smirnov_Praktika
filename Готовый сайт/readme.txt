Задача 15: Создание панели администратора сайта

В корневой папке (NLO) 3 файла, относящихся к этой задаче:

1) admin_panel.php - главная страница панели администратора, на которой расположены ссылки
на страницу редактирования пользователей и страницу редактирования категорий.

2) category_admin_panel.php - страница редактирования категорий, позволяет добавлять и удалять категории, 
а так же переходить на страницу редактирования подкатегорий выбранной категории (конпка-шестеренка, 
страница редактирования подкатегорий - podcategory_admin_panel.php (так же в корневой папке)). Страницы категорий
и подкатегорий могут использовать файл со скриптами, находящийся по адресу "NLO/includes" с названием category_admin_panel_cfg.php.

3) users_admin_panel.php - уже представлена в отчете по работе №14

Как проверить работу на самом сайте:

1.Открыть главную страницу в корневой папке сайта (main.php)
2.В правом верхнем углу нажать "Войти"
3.В поле почты вписать Stark@gmail.com 
4.В поле пароля 12345678
5.Нажать на значок человечка в кружочке в правом верхнем углу
6.Нажать на кнопку "Панель администратора"

Задача 16: Создание форм для создания, редактирования и удаления данных

Формы для работы с категориями представлены в предыдущей работе.

Формы для создания объявлений на сайте:

В корневой папке файл product_new_category.php отвечает за начало создания объявления. Далее происходит переход на product_new.php.
Оба файла используют файл со скриптами, находящийся по адресу "NLO/includes" с названием new_product_cfg.php.

Вывод списка объявлений происходит на главной странице сайта (main.php). Страница main.php использует файл со скриптами,
находящийся по адресу "NLO/includes" с названием main_cfg.php для вывода объявлений и их сортировки по категориям и подкатегориям.

Как проверить работу на самом сайте:

1.Повторить пункты 1-4 из прошлого раздела, чтоб авторизоваться. В теории можно делать и не через администратора, но тогда не будет
доступа к удалению объявлений не с вашего аккаунта.
2.Нажать на кнопку "Выставить товар" на навигационной панели или зайти в личный кабинет и нажать на "Новое объявление"
3.Выбрать категорию товара
4.Заполнить информацию о товаре.
5.Товар будет находится на главном сайте и в разделе "Мои объявления" в личном кабинете. При переходе на его страницу, можно его удалить,
нажав на соответствующую кнопку.

Поиск производится на главной странице. Можно выбрать категорию, после подтверждения можно так же выбрать подкатегорию (работает кривовато,
выбранные значения не сохраняются в полях при подтверждении).
